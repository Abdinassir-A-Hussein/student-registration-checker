# Student Registration Status Checker

## Purpose
This project demonstrates how the same programming problem can be implemented in four different programming languages: Python, Java, JavaScript, and C. It highlights differences in syntax, structure, and programming paradigms.

## Features
- Prompts for student name
- Prompts for number of registered units
- Displays overload status if units exceed 7
- Outputs a clear summary

### Python

cd python
python main.py
